# Modernized Dhrystone 2.1 Benchmark

This is a **modernized version of the Dhrystone 2.1 benchmark** in C, based on the original public-domain code by Reinhold P. Weicker.

## Features

- Fully compatible with **Dhrystone 2.1** verification values.
- Command-line argument support for number of iterations.
- Measures **total execution time** in microseconds and seconds.
- Calculates **Dhrystones per second** and **DMIPS**.
- Warning-free compilation with GCC.
- Works on Linux, WSL, and embedded systems with standard C library.
